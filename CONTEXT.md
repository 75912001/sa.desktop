# 游戏配置

游戏配置定义桌宠客户端使用的宠物、角色、敌人组和托盘菜单数据. 它描述游戏内容和客户端可配置行为, 不描述加载流程或资源解析实现.

## Language

**游戏配置**:
桌宠客户端使用的领域数据集合, 包含宠物、角色、敌人组和托盘菜单等可配置内容.
_Avoid_: 运行时配置

**宠物模板**:
配置表中定义的一种宠物类型, 包含基础属性、成长、元素、技能槽、描述和动画帧配置. 它不是玩家或战斗中实际存在的一只宠物对象.
_Avoid_: 宠物类, 实际宠物对象

**宠物模板 ID**:
用于唯一标识宠物模板的配置 ID. 敌人组中的敌人条目引用宠物模板 ID, 不是引用独立的敌人模板.
_Avoid_: 敌人模板 ID

**稀有度**:
宠物模板中表示稀有等级的字段.
_Avoid_: 品质

**技能模板 ID**:
用于唯一标识技能模板的配置 ID. 宠物模板的技能槽引用技能模板 ID.
_Avoid_: 配置 ID

**技能模板**:
配置表中定义的技能类型, 可被宠物模板的技能槽引用. 它不是战斗中一次具体的技能释放.
_Avoid_: 技能实例, 技能释放

**技能槽**:
宠物模板上用于引用技能模板的固定位置. 空槽使用 0 表示, 因此它不是只包含有效技能的可变列表.
_Avoid_: 技能列表

**动作帧表**:
配置表中把方向、武器类型和动作映射到帧 ID 序列的数据. `sprite` 是配置源文件中该数据的既有字段名, `action_frame_entries` 是代码中结构化后的内存集合字段名; 领域讨论时统一称为动作帧表, 它不是 PNG 图集, 也不是已经构建好的播放器动画资源.
_Avoid_: 动画资源

**方向**:
动作帧表中用于区分角色或宠物面向的八向分类, 如 down、left、up 和 downright.
_Avoid_: 朝向

**动作**:
动作帧表中用于区分行为语义的分类, 如 attack、stand 和 walk. 它不是已经构建好的动画资源.
_Avoid_: 动画动作

**帧 ID**:
动作帧表中引用的数字帧标识, 对应 `.tpsheet` 中的帧文件名. 它用于把配置动作和图集切帧数据连接起来.
_Avoid_: 帧编号

**元素分布**:
宠物模板上 earth、water、fire、wind 四个元素的数值分配. 它不是单一元素标签.
_Avoid_: 元素

**基础属性范围**:
宠物模板上描述生命、攻击、防御和敏捷等基础属性取值区间的数据.
_Avoid_: 数值属性

**成长范围**:
宠物模板上描述生命、攻击、防御和敏捷成长值区间的数据.
_Avoid_: 数值属性

**战斗倍率**:
宠物模板上描述暴击、反击、闪避、命中、暴击伤害加成和异常状态抗性等战斗比率的数据.
_Avoid_: 概率属性, 派生属性

**默认战斗倍率**:
宠物配置中为宠物模板提供的默认战斗倍率值. 它不包含生命、攻击、防御和敏捷等基础属性范围.
_Avoid_: 默认属性

**栖息地**:
宠物模板中描述该宠物通常出现地点的信息.
_Avoid_: 来源信息

**1级出生地**:
宠物模板中描述 1 级宠物获取或出生地点的信息.
_Avoid_: 来源信息

**角色模板**:
配置表中定义的一种角色类型, 包含角色名称、颜色、角色标记和动画帧配置. 它不是玩家账号下实际存在的角色对象.
_Avoid_: 角色外观模板, 实际角色对象

**角色模板 ID**:
用于唯一标识角色模板的配置 ID.
_Avoid_: 配置 ID

**玩家角色标记**:
角色模板上用于标识该模板是否代表玩家角色类型的布尔字段.
_Avoid_: 是否角色

**角色颜色**:
角色模板上用于区分同类角色颜色变体的标记.
_Avoid_: 颜色, 菜单颜色

**武器类型**:
角色模板动作帧表中用于区分不同武器类型动作集合的分类, 如 unarmed、axe、bow、spear 和 throw. 它不是玩家持有的一件具体装备.
_Avoid_: 武器, 装备实例

**敌人组**:
战斗中用于生成敌方单位的一组配置, 包含敌人数量、等级范围、是否 Boss、捕获规则和可出现的宠物模板列表.
_Avoid_: 遭遇组

**敌人组 ID**:
用于唯一标识敌人组的配置 ID.
_Avoid_: 配置 ID

**敌人条目**:
敌人组中的单条敌人配置, 引用一个宠物模板 ID, 并可声明权重和等级. 它不是战斗运行时已经生成的敌方单位.
_Avoid_: 敌人实例

**敌人数量范围**:
普通敌人组中用于随机决定敌方单位数量的最小值和最大值.
_Avoid_: 数量范围

**敌人等级范围**:
普通敌人组中用于随机决定敌方单位等级的最小值和最大值.
_Avoid_: 等级范围

**角色等级偏移范围**:
普通敌人组中基于玩家角色等级计算敌人等级时使用的最小偏移和最大偏移.
_Avoid_: 等级偏移, 范围

**宝宝出现率**:
普通敌人组中控制敌方单位以 1 级宠物宝宝形式出现的概率.
_Avoid_: 宝宝概率, 成为宝宝概率

**可捕获**:
敌人组中表示敌方单位是否允许被捕获的规则.
_Avoid_: 捕获开关

**Boss 组**:
使用固定敌人条目和固定等级生成敌方单位的敌人组.
_Avoid_: 首领组

**必出敌人**:
敌人组中权重为 0 的敌人条目, 会先于加权随机选择进入出场列表.
_Avoid_: 固定敌人

**加权池**:
敌人组中权重大于 0 的敌人条目集合, 用于按权重随机选择额外敌方单位.
_Avoid_: 随机池

**战斗单位**:
战斗运行时生成并显示的单位, 可以来自宠物模板、角色模板或敌人组中的敌人条目.
_Avoid_: 出场单位

**位置编号**:
战斗站位中用于标识一方 0 到 9 号位置的编号.
_Avoid_: 站位编号, 阵位

**托盘菜单配置**:
客户端托盘菜单的可配置结构和显示规则, 包含固定菜单项、样式以及设置窗口中的缩放和透明度状态.
_Avoid_: 菜单模板, 运行时菜单配置

**菜单颜色**:
托盘菜单配置中用于控制面板、边框、文字、高亮和禁用状态的具体颜色值.
_Avoid_: 颜色主题

**菜单项**:
托盘菜单配置中可显示或可点击的单个菜单单元.
_Avoid_: 菜单节点

**缩放选项**:
设置窗口配置中用于选择桌宠窗口缩放比例的滑动选项.
_Avoid_: 显示选项

**透明度选项**:
设置窗口配置中用于选择桌宠窗口内容透明度的滑动选项.
_Avoid_: 显示选项

**菜单命令**:
托盘菜单项触发的客户端行为, 如显示隐藏、鼠标穿透、重置位置、打开选项和退出.
_Avoid_: 菜单动作

**显示名**:
用于 UI 或调试输出展示的名称文本. 它不等同于配置字段名本身.
_Avoid_: 名称

**描述**:
宠物模板、角色模板或技能模板中用于说明该配置内容的文本.
_Avoid_: 说明文本

**动作白名单**:
托盘菜单配置中允许展示或构建的桌宠动作筛选集合. 它不是当前宠物模板最终可播放动作的完整列表.
_Avoid_: 可用动作

**方向白名单**:
托盘菜单配置中允许展示或构建的桌宠方向筛选集合. 它不是当前宠物模板最终可播放方向的完整列表.
_Avoid_: 可用方向
